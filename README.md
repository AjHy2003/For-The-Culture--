# For the Culture 🎌

AI-powered anime, manga, manhwa, novel & lifestyle recommendation engine.

## How to Run

### 1. Install Node.js
Download from https://nodejs.org (LTS version)

### 2. Set your API key
**Windows (Command Prompt):**
```
set ANTHROPIC_API_KEY=your_key_here
```
**Mac/Linux:**
```
export ANTHROPIC_API_KEY=your_key_here
```

### 3. Start the server
```
node server.js
```

### 4. Open in browser
Go to: http://localhost:3000
